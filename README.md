# digitools platform

this is my final project for the class. it's basically a store where you can buy digital stuff, like templates and ui kits, and add them to a shopping cart thingy.

## technologies i guess
- React (vite is fast but weird sometimes)
- Tailwind css
- some icons from lucide
- react-toastify (for the little popup messages that say you bought something)

### features
1. navbar that sticks to the top
2. hero banner with floating stuff
3. products that you can click and buy
4. a shopping cart that works (i think)
* steps section to tell people what to do
- footer with social links and mostly fake text below it

### how to run it
just clone it and do `npm install`. then do `npm run dev` and it should open on localhost 5173 if nothing breaks
